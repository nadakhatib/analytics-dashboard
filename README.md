# 📊 Analytics Dashboard

A responsive analytics dashboard to track website performance, traffic sources, and user behavior.

## ✨ Features

- 📈 Key performance indicators (KPI cards)
- 🌙 Dark / Light mode toggle
- 📱 Fully responsive on all devices
- 🥧 Interactive charts with Chart.js
- 📊 Traffic sources analysis
- 👥 Visitor types tracking
- 🌍 Geographic data (countries & cities)
- 📑 Conversion pages performance
- 📋 Exportable reports

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript
- Chart.js
- Font Awesome
